無料デプロイガイド

サイトを無料で公開する手順

Step 1: GitHubアカウントを作る

github.com/signup でアカウント作成（無料）。
新しいリポジトリを作成 → リポジトリ名は「fuji-soul-tours」でOK。

Step 2: ファイルをアップロード

リポジトリの「Add file」→「Upload files」で以下をアップロード:
index.html, images.js, admin.html

Step 3: Vercelでデプロイ（無料）

vercel.com → Sign Up with GitHub
→ 「Add New Project」→ 先ほどのリポジトリを選択 → Deploy
→ 数秒で fuji-soul-tours.vercel.app のようなURLで公開される！

Step 4: 独自ドメインの設定（任意）

ドメインを買う場合（年間約¥1,500〜）:
→ Namecheap or Cloudflare Registrar で購入
→ 例: fujisoultours.com
→ Vercel の Settings → Domains → ドメインを追加
→ 表示されるDNSレコードをドメイン管理画面で設定
→ SSL証明書はVercelが自動設定（無料）

Step 5: 画像ホスティング（無料）

画像はサイトに直接置かず、CDNで配信がベスト:
→ imgBB — 無料、URLコピーするだけ
→ Cloudinary — 無料枠25GB、画像の自動リサイズも可能
→ そのURLをこの管理画面から設定

💰 コストまとめ

ホスティング（Vercel）: ¥0
SSL証明書: ¥0（Vercel自動）
画像CDN（imgBB/Cloudinary）: ¥0
